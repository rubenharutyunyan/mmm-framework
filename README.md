# mmm-framework

__Bayesian Marketing Mix Modeling framework in Python.__  

Modular, object-oriented package for data preparation, feature engineering, Bayesian modeling (PyMC / PyMC-Marketing), diagnostics, response curves and budget optimization.

__Repo rules__

* pas de data dans le repo
* pas de notebooks en V0 (on fera `examples/` plus tard si besoin)
* code dans `src/mmm/`
* tests dans `tests/`
